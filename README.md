# SmartUrbanAI_progetto
SmartUrban AI" è una Web App didattica che simula un assistente per la mobilità urbana. Include un chatbot AI per info su parcheggi e un sistema di segnalazione posti auto in tempo reale. Obiettivo: sperimentare l'integrazione Web-AI, la progettazione UML e lo sviluppo Agile tramite Git
